# CoGBD: A Consistency-Guided Framework for Universal Graph Backdoor Defense


## Required Dependencies

Please make sure the following dependencies are installed before running the code:

- **NumPy** == 1.26.3  
- **PyTorch** == 2.7.1  
- **PyTorch Geometric** == 2.6.1  
- **Scikit-learn**==1.0.2
- **ogb**
- **deeprobust**

## Quick Start
To reproduce the performance reported in the paper, you can run following the bash file:

```bash
bash script/run.sh
```
